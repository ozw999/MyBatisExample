# MyBatisExample
* 这已不仅仅是一个Mybatis 示例。

* 项目目前使用到 Spring Boot + MyBatis + SQLite + Security + JWT 。

* 包含 play 、 user 、 auth 、login 四个模块。

* 分别用了MyBatis的三种写法以供参考。

* 通过Security进行用户授权认证及登录控制。
